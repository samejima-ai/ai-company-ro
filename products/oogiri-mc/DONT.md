# スコープ外定義 — ヤジ（大喜利MCボット）

以下はこのプロジェクトのスコープ外とする。
AI能力の向上・フェーズ進行に伴い、一部は将来的にスコープ内へ移行する可能性がある。
ただし「## 事業を壊すハード制約」は**例外なく禁止**であり、能力向上でも解除しない。

## 事業を壊すハード制約（例外なく禁止・恒久）

HANDOFF §2 / §5.9 由来。これらは規約BAN・炎上・返金・規制・LTV崩壊の主因であり、道徳論ではなく事業判断として禁止する。
これらは CLAUDE.md（確率的遵守）だけでなく、F4 モデレーション＋ sensors＋ spec/invariants.feature の Evil Path で機械的に二重チェックする。

- 射幸性ゲーミフィケーション（スロット型の煽り・「あと少しで」型の課金誘導）。
- 孤独・依存の搾取。恋愛関係の錯覚による課金誘導。過剰支出の助長。
- 未成年の性化、未成年への色恋・身体・容姿の話題。相手が未成年と疑われる場合は当該領域に踏み込まない。
- 属性（人種/民族/性別/性的指向/障害/宗教/出自等）への攻撃・嘲笑。
- 自傷・自殺・摂食障害・依存の助長や手段の言及。
- 医療・法律・金融の断定的助言。
- 個人情報の収集・特定・暴露。
- 性的・露骨（NSFW）な内容。

### 反面教師（「頂き女子」マニュアル由来の禁止手口）

添付の操作マニュアルは**反面教師**として参照し、その手口を**実装しない**。以下はボット自身が行ってはならない（F4 出口検査・red-team eval の対象）：

- 偽の悩み設定で同情を引き、金銭・見返りを引き出す。
- 「助けたい／守りたい」を意図的に引き出す信頼搾取（いわゆる信頼関係構築コミットの悪用）。
- やみえい（病み営業）：精神的に辛そうな演出で支援・送金を誘発する。
- 未来形色恋・恋人未満の関係性演出による依存形成。
- 連投・過剰課金の助長（→ 逆に「塩対応クールダウン」を正の仕様とする。SPEC F4 / HANDOFF §5.7）。
- 特定の常連を恋人扱いし、贔屓で課金を引き出す。

## MVP（Phase 0）でのスコープ外（後送り）

- 投げ銭・決済（Stripe／暗号決済）、買い切りSKU、月額ロール。→ Phase 1 以降。インターフェースの席のみ用意。
- 投げ銭最優先キュー・高額投げ銭の見せ場演出の本実装。→ Phase 1。
- 音声・アバター・配信映像。
- fine-tune／few-shot 差し替えによる「遅い更新」。→ MVP は封印、速い更新（prompt・辞書・記憶追記）のみ。
- マルチペルソナ、統括AIの自動番組進行、クロスプラットフォーム展開。
- 複数サーバー（ギルド）同時運用。MVP は 1 サーバー。

## dialog-harness 標準のスコープ外

- 創造的UXデザイン（独自UIなし、Discord ネイティブUIで完結）。
- 複雑な状態管理UI。
- パフォーマンス最適化（MVP は機能検証優先）。
- 未知の外部API統合（Discord／LLMプロバイダ以外の新規連携）。

## スコープ外の理由

- ハード制約: 事業の存続条件。違反は一発で炎上・規制・返金・信頼喪失に直結する。
- 反面教師手口: 「操作ではなく芸で稼ぐ」という事業の芯と正面から矛盾する。短期収益と引き換えに LTV と法的・規約リスクを破壊する。
- Phase 0 後送り: 7日リテンションが立つ前に課金導線を付けると、検証対象がぶれ、健全性も担保できない。
- DH標準: 現時点の AI 自律開発で品質を保証しづらい領域。能力向上に伴い縮小する。
